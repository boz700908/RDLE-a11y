1.4 (2026-03-21)
1. Fixed an issue where cursor navigation occasionally freezes (hopefully).
2. Adapted to the new version, fixed an issue where the helper could not open in game version 1.0.5. If you haven't updated the game, the old version should still work with this version of the mod.
3. Added support for editing more properties:
3.1 For certain events that allow specifying a room (e.g. SetTheme), added support for room selection.
3.2 Added support for editing Change Players' Rows events.
3.3 Added preliminary support for editing Set Game Sound events, allowing customization of various in-game sound effects. Note: there are quite a few issues at the moment, so it's best not to use this yet.
3.4 Added support for editing rank max mistakes and rank descriptions.
3.5 There should be a few more, but I can't remember them all right now.
4. When navigating events, the tooltip will now be announced last, which helps quickly understand the important properties of an event.

1.31 (2026-03-12)
1. Fixed a bug where Alt+Shift+R/T did not work as expected.
2. Optimized the announcement for adjusting attribute values using R/T.
3. Fixed an issue with floating-point precision loss.

1.3 (2026-03-07)
1. Added a set of shortcuts to perform simple edits on event properties without opening the helper. (See section 4.5.2 of the manual)
2. Optimized row editing, now supports more properties.
3. When selecting music, you can now choose from all built-in music in the game.
4. Fixed a bug where audio files in the level directory could not be previewed.
5. Added support for rich text tags in UI. If there are links in the currently focused option, you can press space to open the selection menu, then use up/down arrows to select, and press enter to open in the default browser.
6. Creating new events no longer automatically opens the helper.
7. If the localization key for an event name is not found, the internal name will be used by default.

1.1.2 (2026-03-05)
1. Helper now supports pressing space to preview selected sound effects in sound selection.
2. Helper's sound list now displays localized names instead of internal names.
3. Optimized UI navigation by filtering out useless controls.
4. Optimized timeline navigation - event name and event position are now announced in a single message.
PS: What? You're asking if I fixed that occasional freeze issue when browsing with left/right cursor? Do you think this looks like it's been fixed? If any of you can fix it, feel free to submit a PR on my GitHub.

1.1.1 (2026-03-04)
1. Added F2 key as an alternative to Insert. If your keyboard doesn't have an Insert key, you can use F2 instead. The original Insert key still works.
2. When navigating events with left/right arrow keys, if you reach the boundary, the event information will be announced again.

1.1 (2026-03-03)
1. Added ctrl+shift+slash to open the edit cursor jump dialog.
2. Added shift+alt modifier key support for z/x (move event) and comma/period (move edit cursor), with a step size of 1 bar.
3. When pressing enter to jump to an event's position and play, the edit cursor will also be moved to the event's position.
4. Enhanced the browse file feature in the helper - the program will now automatically copy selected files to the level directory.
5. Improved ctrl+v paste functionality, fixed the issue where the beat property remained unchanged after pasting.
6. Improved menu navigation, fixed the issue of browsing to duplicate controls.
7. Fixed a bug where the main menu navigation remained active when entering editor settings from the main menu.
