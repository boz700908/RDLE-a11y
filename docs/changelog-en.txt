1.6.3 (2026-04-21)
Guaranteed compatible version: 1.0.5 (updated 2026-04-21)
1. Added support for setting a custom nurse counting sound effect.
2. All keyboard shortcuts are now fully locked while the helper is open for event editing. To prevent the helper from causing a freeze if something goes wrong, you can press Esc 5 times quickly in-game at any time to force-terminate the helper and unlock the keyboard.
3. When navigating to an event that has a comment at the same position, a sound effect will play and the comment will be read out automatically.

1.6.2 (2026-04-08)
Guaranteed compatible version: 1.0.5 (updated 2026-03-26)
1. Added shortcut action support for some events (e.g. automatically adding nurse voice lines based on the rhythm of a oneshot beat).
2. When selecting a sound, you can now manually enter a filename.
3. Fixed an issue where the sound list did not automatically focus on the currently selected item when opening the helper.

1.6.1 (2026-04-06)
Guaranteed compatible version: 1.0.5 (updated 2026-03-26)
1. Fixed an issue where commas in the original sentence were accidentally split into multiple elements when editing rank descriptions.
2. When an event type is selected in the add event menu, pressing ctrl+enter will automatically create the event and open the helper. Without ctrl, the helper will not open automatically after creating the event.
3. When inserting an event chain, you can now press left/right arrows to adjust the speed multiplier to stretch/compress the spacing between events.

1.6 (2026-04-01)
Note: Starting from this version, the mod will only guarantee compatibility with the latest public beta of the game. The compatible version will be noted in the changelog with each update. The mod may still work in other game versions, but not all features are guaranteed to function correctly. When reporting issues, please make sure you are using the compatible version or above.
Guaranteed compatible version: 1.0.5 (updated 2026-03-26)
1. Added support for the conditions system. (See section 4.7 of the manual)
2. When previewing sounds in the helper, pressing space again on the currently playing audio will stop playback. Closing the helper will also stop all sounds currently being previewed.

1.3 (2026-03-07)

2. Optimized row editing, now supports more properties.
3. When selecting music, you can now choose from all built-in music in the game.

7. If the localization key for an event name is not found, the internal name will be used by default.
